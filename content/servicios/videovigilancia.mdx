# Videovigilancia (CCTV)

Instalamos y mantenemos sistemas de videovigilancia y seguridad completos para empresas, comercios y particulares. Nuestras soluciones incluyen cámaras de alta definición, grabación digital y acceso remoto para tu tranquilidad.

## Tipos de Sistemas

### Cámaras Domésticas
- **Cámaras domo**: Discretas y versátiles para interiores
- **Cámaras bullet**: Robustas para exteriores
- **Cámaras PTZ**: Con movimiento y zoom controlado
- **Cámaras térmicas**: Para detección en condiciones de baja visibilidad

### Sistemas Empresariales
- **Videovigilancia IP**: Alta definición y escalabilidad
- **Sistemas híbridos**: Compatibles con cámaras analógicas e IP
- **Control de acceso**: Integración con sistemas de entrada
- **Analítica avanzada**: Detección de movimiento y reconocimiento

## Características Principales

### Calidad de Imagen
- Resolución hasta 4K Ultra HD
- Visión nocturna infrarroja
- Compresión H.264/H.265 para optimizar almacenamiento
- WDR (Wide Dynamic Range) para condiciones de luz variables

### Almacenamiento y Grabación
- DVR/NVR con capacidad configurable
- Grabación continua o por detección de movimiento
- Retención de datos personalizable
- Backup automático en la nube (opcional)

### Acceso Remoto
- App móvil para iOS y Android
- Acceso web desde cualquier dispositivo
- Notificaciones push en tiempo real
- Control PTZ desde móvil

## Beneficios

- **Seguridad 24/7**: Vigilancia continua de tu propiedad
- **Disuasión**: La presencia de cámaras reduce el riesgo de incidentes
- **Evidencia**: Grabaciones útiles en caso de incidentes
- **Tranquilidad**: Acceso remoto para verificar el estado en cualquier momento
- **Ahorro**: Reducción de costes de seguridad tradicional

## Preguntas frecuentes

### ¿Qué tipos de cámaras de seguridad ofrecen?

Ofrecemos una amplia gama de cámaras: domo para interiores, bullet para exteriores, PTZ con movimiento controlado, térmicas para condiciones especiales y cámaras con visión nocturna. Cada tipo se adapta a necesidades específicas.

### ¿Los sistemas incluyen grabación y almacenamiento?

Sí, todos nuestros sistemas incluyen DVR (Digital Video Recorder) o NVR (Network Video Recorder) con almacenamiento configurable. Puedes elegir la capacidad según tus necesidades de retención de grabaciones.

### ¿Puedo acceder a las cámaras desde mi móvil?

Absolutamente. Todos nuestros sistemas incluyen acceso remoto vía aplicación móvil para iOS y Android, así como acceso web desde cualquier dispositivo con conexión a internet.

### ¿Ofrecen mantenimiento de los sistemas instalados?

Sí, ofrecemos contratos de mantenimiento preventivo y correctivo. Incluyen revisiones periódicas, limpieza, actualizaciones y soporte técnico 24/7 para emergencias.

### ¿Los sistemas son compatibles con otros equipos de seguridad?

Sí, nuestros sistemas se pueden integrar con alarmas, control de acceso, domótica y otros sistemas de seguridad existentes para crear una solución integral.

### ¿Qué pasa si hay un corte de electricidad o internet?

Los sistemas incluyen baterías de respaldo para mantener la grabación durante cortes eléctricos. Para cortes de internet, las cámaras siguen grabando localmente y se sincronizan cuando se restaura la conexión.
